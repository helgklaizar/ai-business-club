# Что произошло: Настройка Монорепозитория и Общей Шапки (Shared UI)

## 1. Настройка Корня (`Craft House`)
* Папка `Craft House` теперь работает как **NPM Workspace** (монорепозиторий). Это позволяет нам создать единую кодовую базу, **не перемещая твои текущие папки и проекты**.
* В корне добавлен `package.json`, который объединяет проекты и заставляет их видеть общие пакеты из соседних папок.

## 2. Создание `shared-ui`
* Мы создали папку `shared-ui` в корне `Craft House`.
* Это пакет-библиотека (`@crafthouse/shared-ui`), внутри которой лежит эталонный UI-компонент `<Header />`. 
* Если ты изменишь шапку здесь, она обновится ВЕЗДЕ. Идеально для консистентного стиля.

## 3. Подключение в `club` (Next.js)
* В твой `package.json` добавлена строчка `"@crafthouse/shared-ui": "*"`.
* В `next.config.ts` добавлена настройка `transpilePackages: ["@crafthouse/shared-ui"]` (позволяет Next.js собирать стили из внешней папки).
* Теперь ты можешь зайти в свои страницы и использовать `<Header />`:
  ```tsx
  import { Header } from '@crafthouse/shared-ui';

  <Header appName="Club" appLink="/">
    ...твоя навигация...
  </Header>
  ```

---
> Это шаг номер один по переходу на общую кодовую базу. Для лендингов у нас есть несколько вариантов: обернуть их в Next.js (что легко и дает хорошую SEO), или написать `<ch-header>` на чистом Javascript.
