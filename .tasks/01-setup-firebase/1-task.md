# Задача: Настройка инфраструктуры (Firebase)

## TL;DR
Подключить Firebase (Auth, Firestore) к проекту. Провайдер оплаты будет выбран и интегрирован позже.

## Контекст
Продукт требует авторизации пользователей и ежемесячной подписки. Firebase будет использоваться как база данных (Firestore) и сервис авторизации. Необходимо инициализировать сервис, собрать ключи доступа и настроить переменные окружения.

## Scope
- ✅ В скоупе: создание проекта в Firebase, настройка `.env.local`, базовая проверка соединения.
- ❌ Не в скоупе: верстка UI авторизации, логика оплаты на фронтенде.

## Ожидаемый результат
В проекте лежат настроенные конфиги Firebase. В консоли нет ошибок при локальном запуске.
