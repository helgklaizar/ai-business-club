# Задача: Админ-Панель (Управление платформой)

## TL;DR
Создание скрытого раздела для администраторов для управления контентом и доступом.

## Контекст
Владелец клуба (администратор) должен иметь возможность добавлять новые кейсы, промпты, записи воркшопов и модерировать базу пользователей без необходимости заходить в сырую консоль базы данных (Firebase Firestore).

## Scope
- ✅ В скоупе: UI/UX админки со списком пользователей и их статусом подписки.
- ✅ В скоупе: CRUD (создание, чтение, обновление, удаление) для Промптов и Кейсов.
- ✅ В скоупе: CRUD для Событий Академии (календарь + видеозаписи).
- ❌ Не в скоупе: Сложные графики аналитики по ROI клуба (оставим на будущее, статистику оплат смотрим в Stripe).

## Ожидаемый результат
Только авторизованные администраторы (со специальной ролью в БД) видят пункт "Админка". В ней можно наполнять базу знаний и проверять статусы резидентов.
